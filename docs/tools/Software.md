## sublime text3

- 支持vue语法高亮

1.进入sublime，选择菜单项“Preferences->Browse Packages...”。创建Vue文件放入。
`见文档docs\.vuepress\public\file\tools\Vue.zip`

2.按下快捷键“ctrl+shift+p”，在打开的packages输入框中输入vue，选择“Set Syntax:Vue Component”进行加载。

3.重启sublime，即能看到语法高亮。
